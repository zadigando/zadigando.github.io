Visão Geral do Projeto

O GZ Artigos Esportivos é uma plataforma de e-commerce desenvolvida com Django, utilizando um sistema de mensageria baseado no RabbitMQ para processar pedidos e pagamentos de forma eficiente.

O fluxo do sistema se baseia na comunicação entre dois serviços independentes: um responsável por gerenciar pedidos e outro por processar pagamentos. A interação entre esses serviços ocorre via RabbitMQ, garantindo escalabilidade e processamento otimizado das transações.

A plataforma conta com funcionalidades como cadastro e autenticação de usuários, carrinho de compras, pagamentos processados de forma assíncrona e gerenciamento de estoque. Além disso, um painel administrativo permite o acompanhamento das vendas e a análise de desempenho do negócio.

O projeto foi desenvolvido com foco em arquitetura distribuída, garantindo maior robustez e desempenho. Para isso, são utilizadas tecnologias como Docker para containerização, REST APIs para comunicação eficiente, RabbitMQ para troca de mensagens entre os serviços e Flask como framework auxiliar para endpoints internos.
